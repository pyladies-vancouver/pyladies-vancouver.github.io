++ SPEAKERS

+ Dr Chris Granade (they/them)
Title: Starting your quantum journey with Python and Q#

If you're interested in getting into quantum computing, Python makes a great environment to start your quantum journey. In this lightning talk, I'll cover some of the skills and resources that can help you spin up on this exciting field using Python, and how you can use Python to write quantum algorithms in Q#, a new quantum programming language from Microsoft.

+Emily Spahn
Title: A Whirlwind Tour of Python Environment Managers

The introduction of Apple’s M1 chip means needing to get previously built tools and programs to run on that chipset. I’ll describe the specific use case I had, which led to an accelerated adoption of a different python environment manager which was already being considered. I’ll review a few options for python environment managers and compare the differences between them.

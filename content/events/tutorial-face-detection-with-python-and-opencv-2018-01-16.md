Ever wanted to learn to do face detection with Python? This tutorial will cover how to use the OpenCV package to work with images and video. Whether you're a complete beginner or a total Python fiend, this tutorial is for you!

For those not attending in person, the tutorial material is here:
https://github.com/pyladies-vancouver/facetrack

Attendees should bring a laptop with an internal camera or a webcam since this tutorial will include plenty of hands-on practice. If you don't have a computer, you'll definitely be able to find a buddy to pair with. Help will be available installing OpenCV with the Anaconda python package manager.

Food will be provided by our sponsor, DemonWare.

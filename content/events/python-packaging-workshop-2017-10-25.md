In this workshop lead by Brett Cannon, we will learn all about Python packaging: what is it for, what are the formats, what tools are available, and what the future holds. We'll go through the process of publishing a package using flit. Bring your own laptop to follow along.

All skill levels welcome!

The workshop is beginner friendly. Being able to edit and save a text file is all you need. We will also use the command line. All the commands will be provided so you can follow along.

Persons with any gender identity are welcome, and should respect that PyLadies Vancouver must be a place where women's voices are centred.
